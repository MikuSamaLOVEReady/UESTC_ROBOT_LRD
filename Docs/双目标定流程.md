# 双目标定

使用matlab对双目图像进行标定

1. 拍摄双目图像然后将左右目图像进行分割。拍摄和分割均在`catpture_photo.py`完成。完成后图片自动保存在文件夹中。
2. 打开matlab

![Untitled](./image/Untitled.png)

点击New Session后Add Images分别添加左右目图像。

点击Calibrate进行标定

1. 标定完成后Export Camera Parameters
2. 获得标定结果
